Flexbox Basics

Flexbox helps us arrange items in a row or column.

display: flex

This turns a container into a flex container.

.container {
  display: flex;
}

Common Values

  • flex-direction: row / column
  • justify-content: start / center / space-between
  • align-items: start / center / end
