Learning the img Tag

We use the img tag to show images on a webpage.

The Basic img Tag

This is the basic structure:

<img src="cat.jpg" alt="A cat" width="300">

src = where the image comes from
alt = text describing the image
width = how wide the image should be

Example 1: Local Image

This example uses an image saved in the same folder as the HTML file.

<img src="dance.jpg" alt="Dancers practicing" width="300">

Example 2: Internet Image

This example uses an image link from the internet.

<img src="https://via.placeholder.com/300x200" alt="Example image" width="300">

Important Things to Remember

  • The img tag shows an image
  • src tells the browser where the file is
  • alt describes the image with words
  • width changes the size
  • The img tag does not need a closing tag
