CSS Quick Cheatsheet

property: value; (explanation)
font-size: 30px; (size of text)
font-weight: bold; (how bold text is)
text-align: left; (where the text is placed)
color: blue; (text color)
background-color: white; (background of element)
margin: 20px; (space outside element)
padding: 15px; (space inside element)
border-radius: 10px; (rounds corners)
list-style-type: square; (changes bullet style)
list-style-position: inside; (bullet position)
border: 2px solid black; (define a border |width|style|color)
