CSS Effects Cheatsheet

filter: grayscale(100%); (makes image black and white)
filter: blur(4px); (makes image blurry)
filter: brightness(150%); (makes image brighter)
filter: contrast(200%); (makes darks and lights stronger)
filter: sepia(100%); (gives image a brown old-photo look)
img:hover { filter: grayscale(100%); } (changes image when mouse is over it)
transform: translateX(100px); (moves the element one hundred pixels to the right)
transform: rotate(90deg) translateY(100px); (rotates the element 90 degrees to the right and moves it down one hundred pixels)
transition: 2s; (makes any change to an element last 2 seconds)
transform: perspective(1500px) translate3d(50px, 100px, -1500px) rotateY(50deg); (make an element change its position in 3D space)
