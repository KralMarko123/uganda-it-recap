HTML + CSS Cheatsheet

Basic Page Structure

<html>
<head>
  <title>Page Name</title>
  <style>
    h1 { color: blue; }
  </style>
</head>
<body>
  Page content here
</body>
</html>

Tag Basics

  • Opening tag → <p>
  • Closing tag → </p>
  • Content → text inside tags
  • Tags use < > brackets

Common HTML Tags

  • <h1> Big title
  • <h2> Section title
  • <p> Paragraph
  • <ul> Bullet list
  • <ol> Numbered list
  • <li> List item
  • <strong> Bold text

Examples

<h1>Title</h1>
<p>Hello world</p>
<strong>Bold text</strong>

<ul>
  <li>Item</li>
  <li>Item</li>
</ul>

Basic CSS

  • color → text color
  • font-size → text size
  • background-color
  • padding → space inside
  • margin → space outside
h1 {
  color: blue;
  font-size: 30px;
}

body {
  background-color: #f0f0f0;
}

Remember

  • HTML = structure
  • CSS = design
  • Most tags open and close
  • Save files as .html
